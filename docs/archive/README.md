# Archive Index

This directory keeps historical Markdown that is no longer part of Seeku's active operating surface.

## Layout

- `root-reports/`
  - one-off verification, scan, and hotfix reports that used to live at the repo root
- `product-history/`
  - older product handoff, incubation, and adjacent-project notes
- `reviews/`
  - historical code review and verification writeups

## Retention Rule

Keep documents here when they still have historical value but should no longer compete with active docs in:

- repo root
- `docs/product/`
- top-level `.planning` anchors

## Current Active Docs

For the current operating surface, use:

- `.planning/STATE.md`
- `.planning/PROJECT.md`
- `.planning/ROADMAP.md`
- `.planning/REQUIREMENTS.md`
- `docs/product/CLI_AGENT_EVAL_HARNESS_2026-04-16.md`
- `docs/superpowers/specs/2026-04-16-cli-search-agent-design.md`
