# Seeku UI设计提示词

你是一个专业的UI/UX设计师。请为 **Seeku**（中文AI人才搜索引擎）设计Web前端界面。

## 产品定位
通过项目证据（代码、开源贡献、项目经历）找到匹配的AI人才，而非简历关键词匹配。

## 目标用户
- 招聘方/HR：重视专业性和可信度
- 技术创始人/工程师：重视代码质量和实际能力
- 中文用户为主

## 核心页面
1. 搜索结果页（筛选器 + 结果列表）
2. 候选人详情弹窗（Evidence Tabs）
3. 管理后台

## 设计参考

### 方案A: Electric Studio（优点）
- 混合主题：深色头部(#0a1628) + 浅色内容区
- 蓝色强调色(#2563eb)专业可信
- 居中大搜索框
- 丰富卡片预览

### 方案G: Professional Navy（优点）
- 双栏布局：左侧筛选器 + 右侧结果
- 海军蓝(#0f172a) + 青绿(#0d9488)商务感
- 功能完整的筛选维度

## 设计要求

**结合两套方案优点，但要有更丰富的视觉层次：**

1. **布局**：双栏（左侧筛选器可折叠 + 右侧搜索+结果）
2. **主题**：混合深浅色，增加渐变或区块层次
3. **配色**：深蓝主色 + 青绿/电光蓝强调 + 暖灰中性色
4. **字体**：中文优化（思源黑体/Noto Sans SC）
5. **风格**：科技感与专业感平衡，有AI前沿感但保持可信
6. **卡片**：微妙阴影/边框层次，悬浮明显反馈，分数突出

## 交互要求
- 筛选器实时筛选，已选条件标签化
- 卡片Hover上浮+阴影
- 详情模态框Tab切换证据
- 响应式：桌面双栏→平板单栏→移动端抽屉

## 输出
1. 设计说明（配色、字体、组件规范）
2. 搜索结果页HTML+CSS（含筛选器和卡片）
3. 候选人详情弹窗HTML+CSS
4. 交互说明

参考风格：Linear.app、Vercel.com、GitHub Cards
