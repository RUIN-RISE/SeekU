## 视觉身份

### 配色方案
```css
:root {
  /* 基础色 */
  --bg-white: #ffffff;
  --bg-light: #fafafa;
  --bg-navy: #0f172a;
  --bg-dark: #0a1628;

  /* 强调色 */
  --accent-blue: #2563eb;
  --accent-cyan: #0d9488;
  --accent-teal: #14b8a6;

  /* 文字色 */
  --text-dark: #0f172a;
  --text-light: #f8fafc;
  --text-muted: #64748b;

  /* UI */
  --glass-bg: rgba(255, 255, 255, 0.8);
  --glass-border: rgba(255, 255, 255, 0.9);
  --card-shadow: 0 4px 20px rgba(0, 0, 0, 0.08);
  --radius: 12px;
  --radius-lg: 24px;
}
```

### 字体
- **中文:** `Noto Sans SC` (400/500/600/700/800)
- **代码/数字:** `JetBrains Mono` (400/500)
- **图标:** Lucide Icons

### 布局原则
- 混合主题：深色头部导航 + 浅色主体内容
- 双栏布局：左侧筛选器(280px) + 右侧结果区
- 毛玻璃效果：卡片使用 `backdrop-filter: blur(12px)`
- 响应式：桌面双栏 → 平板单栏 → 移动端抽屉

## 响应式断点

| 断点 | 布局 |
|------|------|
| > 1024px | 双栏完整显示 |
| 768-1024px | 筛选器折叠为按钮，点击展开 |
| < 768px | 全宽卡片，筛选器底部抽屉 |
